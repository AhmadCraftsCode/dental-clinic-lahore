# Lahore Smile Center

Dental clinic marketing site — Next.js App Router + Tailwind CSS v4.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Home page sections

- Navbar (sticky, WhatsApp CTA)
- Asymmetric hero with illustrated visual panels (icons + patterns)
- Services (editorial list, PKR prices)
- Why choose us
- Testimonials
- Contact CTA + footer
- Floating WhatsApp button (all pages via layout)

## Replace hero visuals with photos

Add images under `public/` and update `HeroVisual.tsx` (or swap panels in `Hero.tsx`) to use `next/image` when ready.
